Utilisateurs AIOStreams

Si vous intégrez cet addon dans AIOStreams, il est fortement recommandé d'augmenter le timeout de l'addon à 25000ms (25s) dans la configuration AIOStreams.
Cela évite que les résultats soient ignorés si les trackers mettent du temps à répondre.

🔒 Trackers UNIT3D (Privés) Recommandé

Ajoutez ici vos trackers basés sur le moteur UNIT3D. UNIT3D est une plateforme de tracker BitTorrent utilisée par de nombreux trackers privés français. L'URL doit être la racine du site (https://site.com). Le token API se génère dans vos paramètres de profil sur le site.
